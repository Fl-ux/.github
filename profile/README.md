# Flux

## About Flux

Flux is a recommendation and review-simulation engine built for localized, culturally aware user experiences.
It models user behavior, retrieves relevant products, and generates personalized, context-rich reviews tailored to Nigerian market realities.

- Purpose: Build multi-dimensional user personas and generate culturally authentic simulated reviews.
- Key components: data pipeline (Parquet outputs), retrieval and recommendation system, backend (FastAPI), frontend (Vite/React), tests.
- Built for the DSN X BCT LLM AGENTS HACKATHON

## Projects

- **Task-A** — User Modeling & Culturally Grounded Review Simulator
  FastAPI + React app with persona modeling, simulation APIs, and end-to-end tests.

- **Task-B** — Hybrid Retrieval-Ranking Engine
  FAISS-based retrieval with contextual filtering and semantic reranking for localized recommendations.
