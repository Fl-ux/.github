# Flux

Flux is a lightweight user-modeling and review-simulation toolkit focused on culturally grounded recommendations (Nigeria-first).
Built for rapid experimentation: data pipeline, FastAPI backend, and React frontend.

- Purpose: Build multi-dimensional user personas and generate culturally authentic simulated reviews.
- Key components: data pipeline (Parquet outputs), backend (FastAPI), frontend (Vite/React), tests.
- Built for the DSN X BCT LLM AGENTS HACKATHON
